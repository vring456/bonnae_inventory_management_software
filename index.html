처음 시작하는 거라면 Supabase 연결이 필요해요.
무료이고 3분이면 설정 완료돼요!

1단계. Supabase 가입
  1. supabase.com 에 접속해서 Start your project 클릭
  2. GitHub 또는 이메일로 회원가입
  3. New Project 클릭 → 프로젝트 이름 입력 (예: 재고관리)
  4. Region: Northeast Asia (Seoul) 선택
  5. 데이터베이스 비밀번호 설정 후 Create new project
  6. 1~2분 기다리면 프로젝트 생성 완료!
supabase.com 열기 →
2단계. 테이블 만들기
  1. Supabase 프로젝트 좌측 메뉴에서 SQL Editor 클릭
  2. New query 클릭
  3. 아래 SQL을 복사해서 붙여넣기
  4. Run 클릭

3단계. API 키 복사
  1. Supabase 좌측 메뉴 하단 Project Settings (⚙️) 클릭
  2. API 탭 클릭
  3. Project URL 복사
  4. anon public 키 복사
